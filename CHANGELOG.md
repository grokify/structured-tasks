## [v0.1.0] - 2026-01-10

### Highlights

- Machine-readable JSON roadmaps with deterministic Markdown generation, modeled after structured-changelog

### Added

- JSON IR schema (v1.0) for machine-readable roadmaps
- `roadmap` package with IR types, parsing, and validation
- `renderer` package with deterministic Markdown generation
- `sroadmap` CLI with `validate`, `generate`, `stats`, and `deps` subcommands
- Two-dimensional categorization: `area` (project component) + `type` (change type)
- Type validation via structured-changelog integration
- Multiple grouping strategies: area, type, phase, status, quarter, priority
- Phased roadmaps with area sub-sections for large projects
- Rich content block support: text, code, diagram, table, list, blockquote
- Table of contents with progress counts
- Overview table summarizing all items
- Version history tracking
- Dependency tracking between items with Mermaid/DOT graph generation
- Customizable status legend
- JSON Schema for IR validation
- Example roadmaps: minimal, full

### Documentation

- README with usage guide and examples
- PRD documenting schema design decisions

### Tests

- Unit tests for core functionality
